# HTML & CSS Lab Projects

This repository contains HTML, CSS, and JavaScript lab assignments and mini-projects created as part of a web development practice portfolio. The focus is on building responsive static web pages, navigation menus, interactive forms, and themed project pages.

## What’s included

- **Pharmacy website**
  - A homepage layout for a pharmacy shop
  - A sticky navigation bar with hover dropdown menus
  - A lab test booking form for patient sample collection
- **Karwachauth project**
  - A themed page featuring a moon and star design
- **Diwali project**
  - A festival-themed HTML/CSS/JavaScript page
- **Tiranga / Indian flag project**
  - A tricolor-themed page with visual styling and music
- **HTML basics demos**
  - Several HTML element examples in `html-basic/`
  - Demonstrations for forms, tables, lists, images, scripts, and more
- **Additional sample pages**
  - `article.html`, `labTest.html`, `pharmacy.html`, `gmail-homePage.html`, and others
  - Exposure to page layout, CSS styling, and interactive UI elements

## Project structure

- `index.html` and project-specific HTML files at the repository root
- `*.css` files for site styling and layout
- `project-diwali/`, `Karwachauth/`, `tri-color/`, and `worldcup-trophy/` folders for themed projects
- `html-basic/` for basic HTML demonstrations and learning examples
- `Images/` contains assets used across pages

## Technologies used

- HTML5
- CSS3
- JavaScript (basic interaction and page behavior)

## How to view the projects

1. Open the repository folder in your browser or code editor.
2. Open any `.html` file in your browser to view the page.
3. For interactive pages, use a live server extension or simply open the HTML directly.

## Notes

- Each page is designed as a standalone demonstration.
- Files use simple styling and layout, making them easy to customize.
- This collection is a good reference for learning HTML/CSS fundamentals.
